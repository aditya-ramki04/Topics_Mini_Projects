# Topics Mini Projects

This repository contains a collection of mini-projects completed as part of coursework focused on data collection, signal processing, and machine learning workflows. Each project builds progressively from raw data acquisition to full classification model development and evaluation.

The goal of this repository is to demonstrate the complete machine learning pipeline:
- Data collection
- Data preprocessing
- Feature engineering
- Model training
- Performance evaluation

---

## 📁 Project 1A – Data Collection & Exploration

**Objective:** Collect and analyze raw data.

In this project:
- Raw sensor/data samples were collected
- Data was cleaned and structured
- Exploratory Data Analysis (EDA) was performed
- Data distributions and trends were visualized

This project establishes the foundation for later modeling tasks by ensuring the dataset is properly formatted and interpretable.

🔗 Link:
https://github.com/aditya-ramki04/Topics_Mini_Projects/tree/main/Project_1A

## 📁 Project 1B – Feature Engineering & Preprocessing

**Objective:** Transform raw data into model-ready features.

In this project:

- Structured datasets were prepared for machine learning models
- Preprocessing pipelines were refined

This phase converts raw inputs into numerical representations suitable for supervised learning.
